# XModZ

### *C++-Based Plugin Platform for PUBG Mobile Lite*

**XModZ** adalah platform eksperimental berbasis C++ yang dirancang untuk menghadirkan **mod menu canggih** di PUBG Mobile Lite. Proyek ini memungkinkan pengguna mengakses fitur-fitur premium dengan cara yang mudah, aman, dan efisien.

## Visi dan Tujuan

::: tip Insight Teknis
Banyak tools mod tradisional membutuhkan akses Root yang berisiko. **XModZ** menyediakan alternatif yang fleksibel untuk pengguna yang ingin memodifikasi permainan tanpa risiko keamanan akibat rooting permanen.
:::

**Solusi XModZ:**
* **Akses Hybrid:** Mendukung integrasi dengan Android melalui metode aman tanpa Root, sambil tetap kompatibel dengan perangkat yang sudah di-root.
* **Mudah Digunakan:** Antarmuka yang intuitif memudahkan pengguna untuk mengakses mod menu tanpa pengalaman teknis khusus.
* **Struktur Terpadu:** Memastikan eksekusi dan kontrol fitur mod yang konsisten di berbagai versi PUBG Mobile Lite.

## Fitur Utama

* **Mod Menu Canggih:** Menghadirkan kontrol lengkap atas gameplay PUBG Lite melalui plugin berbasis C++.
* **Ekosistem Plugin Terbuka:** Mendukung pengembangan plugin pihak ketiga dengan API standar untuk memperluas fungsi mod.
* **Integrasi Aman:** Desain yang memperhatikan keamanan perangkat, sehingga risiko banned atau crash bisa diminimalkan.
* **Kompatibilitas Luas:** Dirancang agar stabil di berbagai perangkat Android, termasuk yang non-rooted maupun rooted.

## Kompatibilitas Sistem

XModZ mendukung berbagai kondisi perangkat:
* **Perangkat Non-Root:** Plugin bekerja via mekanisme aman tanpa Root.
* **Perangkat Rooted:** Memaksimalkan performa dengan kontrol penuh.
* **Dukungan Universal:** Stabil di berbagai versi PUBG Mobile Lite melalui implementasi API yang konsisten.

## Roadmap Pengembangan

XModZ adalah proyek yang terus berkembang. Fokus kami adalah mengeksplorasi teknologi C++ terbaru untuk memastikan platform tetap ringan, cepat, dan mudah diakses oleh komunitas modding global.

> **Eksperimen Tanpa Batas.** Kontrol PUBG Lite dengan cara modern, aman, dan berbasis C++.
