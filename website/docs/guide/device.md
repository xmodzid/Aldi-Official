# XModZ Mod Plugin Compatibility

**XModZ** hanya mendukung perangkat Android dengan **ABI `armeabi-v7a`**.  

---

## ⚠️ Kompatibilitas

| Perangkat / ABI      | Status          |
|---------------------|----------------|
| armeabi-v7a          | ✅ Didukung     |
| arm64-v8a            | ❌ Tidak Didukung |
| x86                  | ❌ Tidak Didukung |
| x86_64               | ❌ Tidak Didukung |

> Pastikan perangkat kamu menggunakan **armeabi-v7a**, jika tidak plugin tidak akan berfungsi.
