A: **ImGui (Immediate Mode GUI)** adalah sistem antarmuka grafis ringan yang sering dipakai dalam tools, overlay, dan aplikasi berbasis native.

**Q: Apakah semua plugin ditujukan untuk game?**

- Fitur tambahan untuk testing atau pengembangan
