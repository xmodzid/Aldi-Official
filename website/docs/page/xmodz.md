**XModZ Plugin** adalah platform **plugin mod Android** berbasis **C++** dengan tampilan **ImGui**, yang dirancang untuk menjalankan modul mod di **PUBG Mobile Lite** dengan **stabilitas tinggi** dan **kinerja optimal**.  

Plugin ini cocok untuk pemain yang ingin pengalaman bermain lebih maksimal tanpa ribet, terutama untuk device 32-bit (armeabi-v7a).

---

## **Arsitektur Support**
- armeabi-v7a / 32 Bit

---

## **Fitur Utama**
- ESP (Player/World)  
- BT Auto Headshot  
- Real High Damage  
- Sky Fly  
- Speed Flash  
- Unlimited Ammo  
- Single Magazine & Double

## **Teknologi & Basis**
- **Bahasa:** C++  
- **Implementasi Library ke Sistem Pubg Lite**
- **GUI / Interface:** ImGui (Immediate Mode GUI)  
- Dirancang modular untuk menambahkan plugin mod tambahan dengan mudah  
- Optimal untuk performa rendah / device 32-bit  
- Tidak Memerlukan Root
- Mudah Di Pakai dan Tidak Ribet, Seperti Game Guardian
