# AxManager

**AxManager** adalah aplikasi Android untuk kontrol lebih dalam terhadap sistem dan aplikasi.  
Berbeda dengan root manager lain, AxManager fokus pada **ADB/Non-Root**, namun tetap mendukung **Root** jika tersedia.  
[Pelajari lebih lanjut](https://fahrez182.github.io/AxManager/)

---

## ‚ú® Fitur Utama
- **Shell Executor**  
  Jalankan perintah shell langsung dari aplikasi. Mendukung **ADB/Non-Root**; Root opsional.  

- **Plugin (Unrooted Module)**  
  Kelola modul pihak ketiga tanpa root.  
  [Info Plugin](https://fahrez182.github.io/AxManager/plugin/what-is-plugin.html)  

- **WebUI (Unrooted)**  
  Jalankan perintah shell via web interface interaktif.  
  [Pelajari WebUI](https://fahrez182.github.io/AxManager/plugin/what-is-plugin.html#webui)
